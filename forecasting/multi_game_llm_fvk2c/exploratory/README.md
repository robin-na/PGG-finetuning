# Multi-Game Exploratory

This subfolder contains non-mainline analysis scripts for the multi-game delegation benchmark.

These scripts are kept for:

- paper-style reproduction analyses
- payoff-simulation side analyses
- older auxiliary plots that are not part of the main benchmark run path

The active benchmark pipeline remains in the parent folder:

- [`../build_batch_inputs.py`](../build_batch_inputs.py)
- [`../parse_outputs.py`](../parse_outputs.py)
- [`../evaluate_outputs.py`](../evaluate_outputs.py)
- [`../analyze_vs_human_treatments.py`](../analyze_vs_human_treatments.py)
- [`../compare_models_with_noise_ceiling.py`](../compare_models_with_noise_ceiling.py)
