# Twin PGG Ablation WD Comparison

This comparison includes baseline GPT-5-mini, full Twin sampled GPT-5-mini, and all six Twin PGG ablation variants. Lower Wasserstein distance is better.

Files:
- `wd_metric_comparison_all6.csv`: long-format per-metric WD summary.
- `wd_metric_matrix_all6.csv`: wide metric table.
- `wd_overall_all6_summary.csv`: average across the six usual WD metrics.
- `run_availability.csv`: completed/evaluated status and parse counts.
- `wd_by_metric_all6.png`: grouped metric plot with all six ablations.
- `wd_overall_all6.png`: overall average WD plot with all six ablations.

Compatibility aliases are also written with `_available` names for older references.
