# Micro Behavior Analysis Outputs

This folder contains analysis artifacts generated by:

- `python Micro_behavior_eval/analysis/run_analysis.py ...`

Each subfolder is one analysis run (`<analysis_run_id>`), containing:

- `metrics_overall.csv`
- `metrics_by_round.csv`
- `metrics_by_game.csv`
- `row_level_scored.csv`
- plot PNG files
- `analysis_manifest.json`

Comparison runs additionally include:

- `comparison_overall.csv`
- `comparison_by_round.csv`
- `comparison_by_game.csv`
- `comparison_overall_errorbars.csv`
- `comparison_by_round_errorbars.csv`
- `comparison_significance.csv`
- `comparison_summary.md`

For benchmark variant layout, typical eval roots are:

- `outputs/benchmark/runs/benchmark_filtered/micro_behavior_eval/no_archetype/<run_id>/`
- `outputs/benchmark/runs/benchmark_filtered/micro_behavior_eval/random_archetype/<run_id>/`
- `outputs/benchmark/runs/benchmark_ood/<factor>/<direction>/micro_behavior_eval/oracle_archetype/<run_id>/`
- `outputs/benchmark/runs/benchmark_ood/<factor>/<direction>/micro_behavior_eval/retrieved_archetype/<run_id>/`
