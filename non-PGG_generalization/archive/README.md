# Archive

This folder holds historical work that is preserved for reference but is not part of the active forecasting pipeline.

Contents currently include:

- `legacy_demographicsOnly/`
  - older demographic-only prototype pipeline work
- `pgg_transfer_eval/`
  - older PGG transfer experiments and evaluation scripts
- `pgg_archetype_transfer/`
  - older PGG-specific transfer outputs
- `twin-20k-500/`
  - older Twin data/assets not used by the active pipeline
- `paper/`
  - draft paper assets and generated LaTeX outputs

Rules:

- do not import from `archive/` in active pipeline code
- if something in `archive/` becomes active again, move it back onto the main path deliberately and update the docs
