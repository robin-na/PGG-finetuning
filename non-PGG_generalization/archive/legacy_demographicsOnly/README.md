# Legacy Demographics-Only Prototype

This folder archives the original PGG-to-Twin transfer prototype.

Characteristics of this legacy setup:

- Twin participant matching was based on only three demographic dimensions:
  age, sex, and education.
- Retrieval mapped Twin demographics into PGG archetype embedding space with a
  simple D-only Ridge model.
- Twin wave-split data was loaded directly from Hugging Face.
- This code is archived as a baseline and is not the intended foundation for the
  richer full-profile PGG-transfer pipeline.
