# Twin-2k-500

`Twin-2k-500` is a benchmark dataset from the paper
"From Product Features to Consumer Choices and Explanations:
A Prompting Framework and Benchmark Dataset for Digital Twins of Consumers."

In short, it provides paired product-choice behavior and natural-language explanations,
designed to evaluate "consumer digital twins" under controlled, scalable experiments.

Source:
- https://pubsonline.informs.org/doi/10.1287/mksc.2025.0262
