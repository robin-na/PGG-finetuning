# Three Diverse Motives for Information Sharing Dataset

## Citation
Vellani, V., Glickman, M., & Sharot, T. (2024). Three diverse motives for information sharing. *Communications Psychology*, 2, 144.

## Source
- **GitHub**: https://github.com/affective-brain-lab/-How-people-decide-when-to-inform-others
- **OSF** (preregistration): https://osf.io/ecxsr/?view_only=bd9c51b1d58149ab9901dfd842d33dc8
- **DOI**: https://doi.org/10.1038/s44271-024-00144-y

## Recruitment
- **Platform**: Prolific
- **Age**: 18-60 (Experiments 1 and replication)
- **Sample**: Information-seeking n=122; information-sharing n=125; 55 completed both
- **Payment**: 7.50 GBP/hr (participation payment)

## Payment Structure (Non-IC)
Hourly participation pay. The information-sharing task includes a **non-instrumental block** where recipients cannot use the information and the paper states "no monetary incentive to share information"---making it a clean non-IC subset for behavior prediction.

## Design
- **Task type**: Multi-trial information seeking/sharing decisions about stocks
- **Trials**: 44 trials + 4 catch trials per block; memory checks; counterbalanced blocks
- **Sharing scale**: 0-6 (how much information to share)
- **Key manipulation**: Instrumental vs non-instrumental sharing blocks
- **Multiplayer**: Not real-time; framed as sharer-to-future-recipients

## Key Variables
- **Trial-level**: Valence of stock value, uncertainty (100 - algorithm accuracy), instrumentality indicator, sharing/seeking rating (0-6)
- **Participant-level**: Fitted beta weights for valence, uncertainty, instrumentality; cluster membership (3 clusters)
- Maps to economic constructs: value sensitivity, ambiguity/uncertainty tolerance, instrumental utility weighting

## Demographics
Age, gender (self-reported). Ethnicity explicitly not collected. Education/income unknown.

## Ethics
UCL departmental ethics committee. Informed consent obtained.
